# 🛒 Retail Sales Machine Learning Project

## 📌 Overview
This project focuses on building a machine learning model using a retail sales dataset. It includes data preprocessing, feature engineering, and predictive modeling.

## 🎯 Problem Statement
- Predict sales amount using regression
- Classify profit as high or low

## 📊 Dataset
- Order ID
- Order Date
- Region
- Product Category
- Sales Amount
- Quantity
- Profit

## 🛠️ Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

## ▶️ How to Run
1. Open the notebook
2. Run all cells
