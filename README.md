# 2.💳 Loan Approval Prediction using Machine Learning

## 📌 Overview

This project focuses on predicting whether a loan application will be approved or rejected using machine learning techniques. It demonstrates data preprocessing, feature engineering, and classification model building.

---

## 🎯 Problem Statement

The goal of this project is to build a model that predicts:

* **Loan Approval Status**

  * 1 → Approved
  * 0 → Rejected

---

## 📊 Dataset Description

The dataset contains information about applicants with the following features:

* Age
* Income
* Credit Score
* Loan Amount
* Approval Status (Target Variable)

---

## ⚙️ Project Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Building
5. Model Evaluation

---

## 🧠 Algorithms Used

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Tree

---

## 📈 Evaluation Metric

* Confusion Matrix
* Accuracy Score

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

---

## ▶️ How to Run

1. Clone the repository
2. Open the notebook (`loan-approval-prediction.ipynb`)
3. Run all cells step by step
